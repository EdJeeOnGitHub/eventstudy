EventStudy Class¶

Event Study core object. Implement the classical event study methodology 1 for a single event. This implementation heavily rely on the work of MacKinlay 2.

  • Run the event study

  • Import data

  • Retrieve results

  • References

Run the event study¶

eventstudy.EventStudy

Low-level way of runing an event study.

eventstudy.EventStudy.market_model

Modelise returns with the market model.

eventstudy.EventStudy.constant_mean

Modelise returns with the constant mean model.

eventstudy.EventStudy.FamaFrench_3factor

Modelise returns with the Fama-French 3-factor model.

Import data¶

eventstudy.EventStudy.import_FamaFrench

Import returns from csv file to EventStudy Class parameter.

Import returns from csv file to EventStudy Class parameter.

eventstudy.EventStudy.import_returns_from_API

Retrieve results¶

eventstudy.EventStudy.plot

Plot the event study result.

eventstudy.EventStudy.results

Give event study result in a table format.

References¶

1

Fama, E. F., L. Fisher, M. C. Jensen, and R. Roll (1969). “The Adjustment of Stock Prices to New Information”. In: International Economic Review 10.1, pp. 1–21.

2

Mackinlay, A. (1997). “Event Studies in Economics and Finance”. In: Journal of Economic Literature 35.1, p. 13.
